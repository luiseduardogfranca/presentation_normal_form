Cálculo Lambda λ

O que é?

 Sistema formal baseado em uma notação para funções;

 É Turing-completo;

 Modelo de computação base de programação funcional;

 Possui dois conceitos: function abstraction e function application;

f() →

Operador térnario com λ

<‍cond> ? <‍expr1>:<‍expr2>

Sintaxe

λ<‍name>.<‍body>

Primeiro definimos a função para true e false

def true = λa.λb.a

def false = λa.λb.b

O equivalente em programação...

              
def true(a):
  def _(b):
    return a
  
  return _


def false(a,b):
  def _(b):
   return b             

  return _

Depois é montado toda a expressão

def cond = λa.λb.λc.((c a) b)

O equivalente em programação...

              
def cond(a):
  def _(b):
    def conditional(c):
      return c(a, b)              
    
    return conditional 

  return _
